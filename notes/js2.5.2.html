Append Prepend

  • Append will insert the thing to last
  • Prepend will insert the thing to first
  • Append and append child works same

Task

  • Create a h1 (name) in the div
  • Create a button (click me) in the div
  • Do these task with the help of js only

Code

                <div>

                </div>
            
                <script>
                    let mydiv = document.querySelector("div")

                    let heading = document.createElement("h1")
                    let button = document.createElement("button")

                    heading.innerText = "sparsh"
                    button.innerText = "Click me"

                    mydiv.append(heading)
                    mydiv.append(button)
                </script>

Output

Code

                <div>

                <script>
                    let mydiv = document.querySelector("div")

                    let heading = document.createElement("h1")
                    let button = document.createElement("button")

                    heading.innerText = "sparsh"
                    button.innerText = "Click me"

                    mydiv.prepend(heading)

                </script>
