Deep copy and shallow copy

Deep copy

Different Ghar

Example:
let obj1 = {
    x:100;
    y:200;
}
obj2={...obj1}

RAM

4k 5k
obj1 obj2

If we change anything to obj1 it will not affect to obj2 because of different address.

Shallow copy

Same ghar

Example: 
let obj1 = {
    x:100,
    y:200
}
let obj2 = obj1;

RAM

obj1
4k
obj2

If we change anything to obj1 it will automatically change to obj2 because of same address
