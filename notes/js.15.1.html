Objects

  • Stores the multiple details like fname, lname, age, salary, other details, ismarried

Syntax

object = { key : value }

Example

            mydetails = {
                fname : "sparsh",
                lname : "gupta",
                age : 24,
                salary : 10000,
                ismarried : false,
                marks : [10,20,30],
                gradde : 'A',
                task : function(){
                    console.log("student")
                },
                otherdetails : {
                    hometown : "delhi",
                    education : "MBA"
                }
            }

How to access properties

Dot Notation

console.log(mydetails.grade)

Output: A

Square Notation

console.log(mydetails['ismarried'])

Output: false

Loops

Only for in loop apply on this.

            for(let maal in mydetails){
                console.log(maal)
            }

Here the maal contain all keys.

            for(let maal in mydetails){
                console.log(mydetails[maal])
            }

Now this will give the values.

            for(let maal in mydetails){
                console.log(maal,mydetails[maal])
            }

Now this will give both key and value.

Access the task function

console.log(mydetails.task())

Mutability check

            mydetails.sex = "male";
            console.log(mydetails)

the new key has been added hence, the objects are mutable.

Value change check

            mydetails.salary = 1000;
            console.log(mydetails.salary)
            Output: 1000

In JavaScript, when you change the value of an existing key in an object, it overwrites the previous value rather than creating a new key.
