Element selection

querySelector

  • It catch the first element it found
  • Syntax-
    document.querySelector("element")
    element: can be h1, p, button, input, etc
    For id
    document.querySelector("#id")
    For class
    document.querySelector(".class")

Example 1

Code

                <h1>Sparsh</h1>
                
                <script>
                    let heading = document.querySelector("h1")
                    console.log(heading);
                </script>

Output on browser

Sparsh

Output on console

<h1>Sparsh</h1>

Example 2

Code

<button onclick="pink()">click me to make me pink</button>

<script>
    let btn=document.querySelector("button")
    function pink(){
         btn.style.color="pink"
         btn.style.backgroundColor="black"
    }
</script>

Output

querySelectorAll

  • It catch all the element it found
  • Syntax-
    document.querySelectorAll("element")
    elements: can be many h1, many p, many button, many input etc

Example

Code

                <h1>Sparsh</h1>
                <h1>Gupta</h1>

                <script>
                    let headings= document.querySelectorAll("h1")
                    console.log(headings);
                </script>

Output on Browser

Sparsh

Gupta

Output on Console

NodeList(2) [h1, h1] <= nodelist is like a array which contain multiple elements
  • We can access nodelist with the help of index like 0 idx will catch the first element, etc.... And with the help of this we can apply the loops as well
  • Example:

                            }
