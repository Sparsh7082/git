Primitive and non primitive data type

Primitve data type

Value

It pass the value

Non primitive data type

Function, array, object

reference - address - ram address

It does not pass the value it pass the ram address

Example: 
let obj1 = {
    x:100,
    y:200
}
let obj2 = obj1;

Here the ram address is passed like if we change something to obj1 then the obj2 will automatically update. Which makes it as shallow copy.
