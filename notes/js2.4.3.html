Element selection

html (getElementBy)

getElementById

Code

                    <h1 id="heading">Sparsh</h1>

                    <script>
                        let heading = document.getElementById("heading")
                        console.log(heading);
                        
                    </script>

Output on Browser

Sparsh

Output on console

<h1 id="heading">Sparsh</h1>

getElementsByTagName

Code

                    <h1>Sparsh</h1>
                    <h1>Gupta</h1>
                    <script>
                        let heading = document.getElementsByTagName("h1")
                        console.log(heading);

                    </script>

Output on Browser

Sparsh

Gupta

Output on Console

HTMLCollection(2) [h1, h1] same like nodelist
  • If the html give the array like thing (because of multiple element selection) then it gives the HTML Collection
  • Element selection => console.log(heading[0])

getElementsByClassName

Code

        <h1 class="heading">Sparsh</h1>
        <h1 class="heading">Gupta</h1>
        
        <script>
            let heading = document.getElementsByClassName("heading")
            console.log(heading);
        </script>

Output on Browser

Sparsh

Gupta

Output on Console

HTMLCollection(2) [h1.heading, h1.heading]
