Fetch

  • Fetch is an API which written in JS native (means the part of JS)
  • It helps to pick the data from any URL or update the data by hitting on the URL
  • API - url - hit - data
  • Fetch works in two steps-
    • Step 1- First it gives bolb data (which we can't able to read)
    • Step 2- Next we need to convert the bolb data to json format (object format) (so that we can read that data)
  • Syntax
    fetch('url')

Async-await (for time)

Example

            async function getdata(){
                let bolbdata = await fetch('https://randomuser.me/api/')    //because we need time to fetch the data
                let actualdata = await bolbdata.json()                      //because we need time to convert the data
                console.log(actualdata)
                console.log(actualdata.results[0].gender)                   //for specific selection
            }
            getdata();

Example 2

            async function getdata(){
                let boldata = await fetch('https://restcountries.com/v3.1/all')
                let actualdata = await boldata.json()
                console.log(actualdata);

            }
            getdata()

            check the output on browser -> right click -> inspect -> console

We can use promise or callback but it is outdated and lengthy so we use async-await only.
