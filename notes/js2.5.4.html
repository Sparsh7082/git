CSS className ClassList

  • These are used to overwrite the class with the help of js.
  • className removes the previous class and add the new class
  • classList add the new class over the previous class which means both the class exist.

Example

Code

    <style>
        .pink{
            background-color: pink;
        }

        .aqua{
            background-color: aqua;
        }
    </style>

    <h1 class="pink">Hello</h1>

    <script>
        let heading = document.querySelector("h1")

        heading.classList.add("aqua")
        console.log(heading.className);
        
    </script>

Output on Browser

Hello

Output on Console

pink aqua

Task

  • When single click add the class pink
  • When double click add the class aqua
  • When scroll remove all the class

Code

    <style>
        .pink{
            background-color: pink;
        }

        .aqua{
            background-color: aqua;
        }
    </style>
    
    <h1>Hello</h1>
    <button>Click me</button>

    <script>
        let heading = document.querySelector("h1")
        let button = document.querySelector("button")

        button.onclick = function(){
            heading.className = "pink"
        }

        button.ondblclick = function(){
            heading.className = "aqua"
        }
        
        button.onwheel = function(){
            heading.className = ""
        }
    </script>

Output

Hello
