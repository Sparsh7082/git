Keyboard Events

  • When we press any key there is some function start running
  • onkeypress (runs -1 idx)
  • onkeydown (runs -1 idx)
  • onkeyup (runs 0 idx)

Example

Code

    <input type="text" onkeypress="keyboardevent()">
    <script>
        let inp = document.querySelector("input")

        function keyboardevent(){
            console.log("I am fired");
            
        }
    </script>

Output on browser

Output on Console

I am fired

Event Object

  • it helps to pick the value
  • Syntax: event.target.value

Example

Code

    <input type="text" onkeypress="eventobj(event)">
    <script>
        let inp = document.querySelector("input")

        function eventobj(event){
            console.log(event.target.value);
            
        }
    </script>

Output on browser

Output on Console

It runs on a -1 idx
