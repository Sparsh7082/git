Settimeout and setinterval

Set time out

After how many milisecond the function should run

Syntax

setTimeout(callback function or handler,time (in miliseconds))

            
Example

            setTimeout(function(){
                console.log("i am running")    
            },2000)                           // 2000 means 2 second

            This code will run after 2 second

set Interval

After every interval the function will run

Syntax

setTimeout(callback function or handler,time (in miliseconds))

            
Example

            setTimeout(function(){
                console.log("i am running")    
            },2000)                           // 2000 means 2 second

            This code will run after every 2 second

            clearInterval(id/rv)       // this will clear the interval with the help of id or return value
