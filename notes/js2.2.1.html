Async Js.

  • The javascript is a single threaded language which means we can do one work at a time. But with the help of Async Javascript we can do mutiple work at a time.
  • Full form of Async Js is Asynchronous Javascript and javascript is a Synchronous language (step by step)
  • We can use the Async Js with the help of
    • Callbacks
    • Promises
    • Async Await
