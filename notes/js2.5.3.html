Innertext InnerHTML

  • Innertext print the text as it is
  • InnerHTML checks the text whether it uses the html element... if it use then it apply it

Example

Code

    <div>
    
    </div>

    <script>
        let mydiv = document.querySelector("div")

        let ul = document.createElement("ul")
        let li = document.createElement("li")

        li.innerText = "<h1>Apple</h1>"
        
        ul.append(li)
        mydiv.append(ul)
    </script>

Output

  • <h1>Apple</h1>

Code

    <div>
    
    </div>

    <script>
        let mydiv = document.querySelector("div")

        let ul = document.createElement("ul")
        let li = document.createElement("li")

        li.innerHTML = "<h1>Apple</h1>"
        
        ul.append(li)
        mydiv.append(ul)
    </script>

Output

  • Apple
